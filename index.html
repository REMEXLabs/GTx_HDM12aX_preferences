Needs and Preference Sets for GTx HDM1.2aX

The purpose of this website is to host a number of “needs and preference sets” or NP sets for use in the course GTx HDM1.2aX. The following NP sets are available (with their most salient features):

  • Anna-UK.json: NP set requiring brown text on a buff background.
    See also a day in the life of … Anna.
  • Lars-NO.json (Norwegian) and Lars-US.json (USA): NP sets requiring subtitles, captions and sign language (depending on context and which of these features is available, i.e. not all at the same time).
    See also a day in the life of … Lars.
  • Monika-DE.json: NP set requiring a bigger font size.
    See also a day in the life of … Monika.
  • Susan-UK.json: NP set requiring subtitles and captions (depending on context, i.e. not both at the same time).
    See also a day in the life of … Susan.
  • Tom-IE.json: NP set requiring an on-screen keyboard.
    See also a day in the life of … Tom.
  • Adam-SV.json: NP set requiring simple Swedish, a simplified user interface and self voicing in applications.
    See also Adam Ljung, a persona created in the AEGIS project. For background information about the simple subtag, see Towards simple language on the web (Tobias Bengfort, 26 June 2017). The definition of “simple” is not straightforward. The language used in the Simple English Wikipedia appears to follow the guidelines for Basic English.
  • Adam-EN.json: the same as the above but with simple English instead of simple Swedish.
  • olb_Alicia_cmn.json: NP set requiring a white-on-black high-contrast theme.
  • olb_Carla.json: NP set requiring a 24 points font size, the link adaptation “emphasizeLinks” and a white-on-black high-contrast theme.
  • olb_KimCallahan.json: NP set requiring American Sign Language.

For the meaning of the needs and preference terms used in these NP set, see the page GPII.eu Common Terms on the main website.
